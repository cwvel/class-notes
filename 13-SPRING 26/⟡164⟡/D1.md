## Optimization overview
- Maximizing/minimizing an objective function
- **Objective function**: $f(x)$, where $x$ are the variables
- Subject to constraints on the variables
	- $C_i(x)$ are **constraint functions** that define certain equations or inequalities that the unknown $x$ must satisfy
	- **Feasible region/set**: $\Omega$
		- Set of variables that satisfies all constraints
		- $\Omega=\{x\in\mathbb{R}^n:C_i(x)=0,i\in\varepsilon,C_i(x)\geq0,i\in I\}$
- **Objective**: scalar function that measures the performance of the system under study
	- We want to find values of variables that satisfies the constraints and optimizes the objective
- Solving steps:
	1. **Modelling**: Identify objective, variables, and constraints
	2. **Solving**: Employing an optimization algorithm to find solutions (usually with computers)
	3. **Analyzing**: Recognizing whether the problem has been successfully solved using optimality conditions
## Linear algebra review
**Vector space*** $V$ is composed of a set of vectors and scalars of field $F$ ($\mathbb{R}$ or $\mathbb{C}$)
1. Vector addition: $\vec x+\vec y\in V$ for $\vec x,\vec y\in V$
	- Associative and commutative
	- $\exists$ "zero vector" $\vec 0\in V$, s.t. $\vec x + \vec 0 = \vec x$
	- $\exists$ $-\vec x\in V$ s.t. $\vec x + (-\vec x)=\vec 0$
2. Scalar multiplication: $a\vec x\in V$ for $\vec x\in V$, $a\in F$
	- Distributive (both scalars and vectors), associative
	- Multipliciative identity ($1\in F$ s.t. $1\cdot\vec x=\vec x\in V$)
	- Additive identity ($0\in F$ s.t. $0\cdot\vec x=0$)
****
A nonempty subset $S$ of vector space $V$ is a **linear subspace** of $V$ if
$$a\vec x+b\vec y\in S$$
for all $\vec x,\vec y\in S,a,b\in F$.
****
Let $M=\{v_1,\dots,v_n\}$ be a set of vectors in $V$. A **linear combination** of vectors in $M$ is
$$c_1v_1+\dots+c_nv_n$$
for $c_i\in F$, $i=1\dots n$. If $M$ is a finite collection of vectors in $V$, the **span** of $M$ denoted by $\text{span}(M)$ is the set of all linear combinations of vectors in $M$.
****
A finite set of vectors $v_1,\dots,v_n\in V$ is
- **linearly independent** $\iff c_1v_1+\dots+c_nv_n=\vec0$ only when $c_i=0$ for all $i=1\dots n$
- **linearly dependent** $\iff$ not all $c_i=0$
****
A finite set of vectors $v_1,\dots,v_n\in V$ forms a **basis** for $V$ if
1. $v_1,\dots,v_n$ are linearly independent
2. $\text{span}\{v_1,\dots,v_n\}=V$
****
Let $X,Y$ be vector spaces. The operator $A:X\to Y$ is a **linear operator** if
$$A(\alpha_1\vec x_1+\alpha_2\vec x_2)=\alpha_1A\vec x_1+\alpha_2A \vec x_2$$
for all $\vec x_1,\vec x_2\in X$, $\alpha_1,\alpha_2\in F$.
If $X=\mathbb{R}^n$, $Y=\mathbb{R}^m$, $A: X\to Y$ can be represented as a $m\times n$ matrix.
****
Let $A:X\to Y$ be a linear operator. The **range** of $A$, denoted by $R(A)$, is
$$R(A):=\{\vec y\in Y:A\vec x=\vec y\quad\text{for some }\vec x\in X\}$$
The **nullspace** of $A$ is
$$N(A):=\{\vec x\in X:Ax=0\}$$
$R(A)$ is a **subspace** of $Y$ and $N(A)$ is a subspace of $X$.
****
A linear operator $P:X\to X$ is a **projection** operator if and only if $P^2=P$.
A projection operator $P$ is called an **orthogonal projector operator** if
$$R(P)\perp N(P)$$
Note: $x=R(P)+N(P)=Px+(I-P)x$
$P(I-P)x=Px-P^2x=0$, $V=R(P)\oplus N(P)$
****
Let $X,Y$ be normed linear spaces with $\|\cdot\|_X$ and $\|\cdot\|_Y$. 
Suppose $A:X\to Y$ is a linear operator. The **operator norm** is defined as
$$\|A\|:=\sup_{\vec0\neq\vec x\in X}\frac{\|A\vec x\|_Y}{\|x\|_X}$$
or equivalently,
$$\|A\|:=\sup_{\vec x\in X,\|\vec x\|_X=1}\|Ax\|_Y$$
## Midterm review

Gradient method.
Considering $f(x)=\frac{1}{2}x^\top Qx-b^\top x$,
$$x^{k+1}=x^k-\alpha_k\nabla f(x^k)$$
steepest descent: $\alpha_k=\arg\min_{\alpha\ge0}f(x^k-\alpha\nabla f(x^k)):=\phi(\alpha)$
- note that $\Rightarrow\phi'(\alpha_k)=0$
or equivalently
$$\alpha_k=\frac{(g^k)^\top g^k}{(g^k)^\top Qg^k}$$
proof of two propositions:
1. $x^{k+1}-x^{k}\perp x^{k+2}-x^{k+1}$
2. $f(x^{k+1})<f(x^k)$
