# Weibull Distribution
Probability of failure: $W$ is the interval necessary to produce the first failure

$$X\sim W(k,\lambda),\qquad k=\text{shape},\quad \lambda=\text{scale}$$
pmf:
$$f(x)=\dfrac{k}{\lambda}\cdot\left(\dfrac{x}{\lambda}\right)^{k-1}e^{-(x/\lambda)^k}\qquad\text{for $x\geq$ 0, and 0 otherwise}$$
:
$$G(x)=\left(\dfrac{x}{\lambda}\right)^k$$
$$G'(x)=\dfrac{k}{\lambda}\left(\dfrac{x}{\lambda}\right)^{k-1}$$
Then
$$f_{X_W}(x)=G'(x)\cdot e^{-G(x)},\qquad x\geq 0$$

cdf:
$$F_X(x)=\int_{-\infty}^x f(x)dx=1-e^{-(x/\lambda)^k}\qquad\text{for $x\geq 0$, and 0 otherwise}$$

### Expected value
$$E(X)=\int_\mathbb{R}xf(x)dx=\int_0^\infty x\left(\dfrac{k}{\lambda}\right)\left(\dfrac{x}{\lambda}\right)^{k-1}\cdot e^{-(x/\lambda)^k}dx$$
Change of variables: $t=\left(\dfrac{x}{\lambda}\right)^k\implies x^k=t\lambda^k\implies x=\lambda t^{1/k}\implies dx=\dfrac{\lambda}{k}\cdot t^{\frac{1}{k}-1}$
$$E(X)=\dfrac{k}{\lambda^k}\int_0^\infty x^k\cdot e^{-(x/\lambda)^k}dx=\lambda\int_0^\infty t^{1/k}\cdot e^{-t}dt$$
Matches the Gamma function $\Gamma(\alpha)\equiv\int_0^\infty t^{\alpha-1}e^{-t}dt$:
$$\boxed{E(X_W)=\lambda\cdot \Gamma\left(1+\dfrac{1}{k}\right)}$$

### Variance
$$\sigma^2(X)=\lambda^2\left(\Gamma\left(1+\dfrac{2}{k}\right)-\left(\Gamma\left(1+\dfrac{1}{k}\right)\right)^2\right)=\dots$$
$$G(x)=\left(\dfrac{x}{\lambda}\right)^k\int_0^\infty x^2\cdot\dfrac{k}{\lambda^k}\cdot\left(\dfrac{x}{\lambda}\right)^{k-1}\cdot e^{-(x/\lambda)^k}dx$$

## Particular cases
$k=1\rightarrow f(x)\equiv \dfrac{1}{\lambda}\left(\dfrac{x}{\lambda}\right)^0\cdot e^-{x/\lambda}=\dfrac{1}{\lambda}\cdot e^{-x/\lambda}\sim\text{Exp}(\alpha=\dfrac{1}{\lambda})$

$k=2\rightarrow$ Rayleigh distribution


# Mixed Discrete and Continuous Distributions
>[!definition]
>Let $X$ be a random variable on $\mathbb{R}$. Then $X$ is of **mixed-type** (discrete and continuous) if there exists:
>i) A discrete part with probability mass function $$(\text{pmf})_i=P(X=x_i)$$ and 
>ii) A continuous part with density function $f(x)$ such that $$P_X(A)=\sum_{x_i\in A}P_i+\int_A f(x)dx$$
>****
>

***Normalization property:*** still sums to 1:
$$\sum_{i=1}P_i+\int_\mathbb{R}f(x)dx=1$$

cdf:
$$F(X_\text{mixed})=\sum_{x_i\in A}P_i+\int_{-\infty}^xf(t)dt$$
