**3 parts of a proof:**
- **Beginning**: Assumptions you are making
- **Middle**: Sentences following logically from what came before
- **End**: What you want to prove is true from the assumptions

**Example**:
Let $n$ be an integer. If $n$ is even, $n^2$ is even.
***Proof***: Suppose $n$ is an even integer. This means we can write $n = 2k$ for some integer $k$. Then $$n^2 = (2k)^2 = 4k^2 = 2(2k)^2$$ Note that $2k^2$ is an integer. Since we could write $n^2 = 2 \cdot j$ for some integer $j$, $n^2$ is even.
$\square$
## Proof methods
### I. Direct proof
To prove $P \Rightarrow Q$:
1. Assume $P$ is true.
2. From this assumption, prove that $Q$ is true.

**Example**: Let $n$ be an integer. If $n$ is odd, then $n+1$ is even.
***Proof:*** Assume $n$ is odd. Then $n=2k+1$ for some integer $k$, $$n^2+1=(2k+1)^2+1=4k^2+4k+2=2(2k^2+2k+1)$$where $(2k^2+2k+1)$ is an integer. Because we can write $n^2+1=2\cdot j$ where $j$ is an integer, then $n^2+1$ is even. $\square$
### II. Contrapositive proof
To prove $P \Rightarrow Q$:
1. Assume that $Q$ is false.
2. Show that $P$ is false.

>**Principle:** "If P then Q" $\Leftrightarrow$ "If not Q then not P''

**Example:** Let $n$ be an integer. If $n^2$ is even, $n$ is even.
***Proof:*** Suppose that $n$ is not even (we have to show that $n^2$ is not even). Then $n$ must be odd, so we can write it as $n = 2k+1$ for some integer $k$. Then: $$n^2 = (2k+1)^2 = 4k^2 + 4k + 1 = 2(2k^2+2k)+1$$
Since $2k^2+2k$ is an integer, we have written $n^2 = 2j+1$ for some integer $j$. So $n^2$ is odd and therefore not even. $\square$
### III. Proof by contradiction
To prove a statement $R$:
1. Assume $R$ is false.
2. Prove a contradiction from this assumption.

**Example:** If $n^2$ is odd, then $n$ is odd.
***Proof:*** Suppose for the sake of contradiction that $n^2$ is odd and $n$ is even. Then $n = 2k$ for some integer $k$. So: $$n^2 = (2k)^2 = 4k^2 = 2(2k^2)$$where $2k^2$ is an integer, therefore $n^2$ is even. So $n^2$ is both even and odd, which is impossible. Hence, if $n^2$ is odd, then $n$ is odd. $\square$

**Example:** Prove that $\sqrt2$ is irrational.
***Proof*:** Suppose $\sqrt2$ is rational. So $\sqrt2=\frac{a}{b}$, where $a,b$ are integers, $b\neq 0$, and $\text{gcd}(a,b)=1$. Squaring both sides, $$2=\frac{a}{b}\implies a^2=2b^2\implies 2\mid a^2 \implies 2\mid a \implies 4\mid a^2=2b^2 \implies 2\mid b$$So $2\mid a$ and $2\mid b$, which contradicts $\text{gcd}(a,b)=1$. Therefore $\sqrt2$ is irrational. $\square$
