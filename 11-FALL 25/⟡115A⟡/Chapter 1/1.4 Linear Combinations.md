>[!definition]
>Let $V$ be a vector space of $F$ and let $S\subseteq V$ be a subset. $v$ can be written as a **linear combination** of vectors in $S$ if there is a finite collection of vectors $v_1,v_2\dots ,v_n\in S$ and scalars $c_1,\dots c_n\in F$ such that $$v=c_1v_1+c_2v_2+\dots +c_nv_n$$
>The $c$'s are called the **coefficients** of the linear combination.

**Ex.** $\begin{pmatrix}3 \\ 5 \\ 2\end{pmatrix}=2\begin{pmatrix}1\\1\\0\end{pmatrix}+\begin{pmatrix}1\\2\\1\end{pmatrix}+\begin{pmatrix}0\\1\\1\end{pmatrix}$
Note that a linear combination of $\begin{pmatrix}3\\5\\2\end{pmatrix}$ can be written in different ways (not necessarily unique):
$\begin{pmatrix}3 \\ 5 \\ 2\end{pmatrix}=2\begin{pmatrix}1\\1\\0\end{pmatrix}+\begin{pmatrix}1\\2\\1\end{pmatrix}$ is another linear combination (the second vector is a LC of the first and third)

Determining if $v\in V$ can be written as a linear combination of vectors in $S$ boils down to determining if a system of equations has a solution.

**Ex.** $\vec{v}=\begin{pmatrix}1\\0\\0\end{pmatrix},\vec{v_1}=\begin{pmatrix}1\\1\\0\end{pmatrix},\vec{v_2}=\begin{pmatrix}1\\2\\1\end{pmatrix},\vec{v_3}=\begin{pmatrix}0\\1\\1\end{pmatrix}$
Then $v$ is not a linear combination of $v_1,v_2,v_3$.

If $v$ is a linear combination of $v_1,v_2,v_3$ then for all $c_1,c_2,c_3\in F$ such that $v=c_1v_1+c_2v_2+c_3v_3$, then they must satisfy:
$\begin{align}c_1+c_2=1\\c_1+2c_2+c_3=0\\c_2+c_3=0\end{align}$
But adding (1) to (3) contradicts (2), so the system has no solutions ($0\neq 1$).

>[!definition]
>Let $S\subseteq V$ be a non-empty subset of vector space $V$. Then the **span** of $S$, $\text{span}(S)$, is the set of all linear combinations of vectors in $S$.
>We also define $\text{span}(\emptyset)=\{\vec{0}_V\}$.
>The span of $S$ is given by $$\text{span}(S)=\{\sum_{i=1}^nc_iw_i:c_i\in F,w_i\in S, n\in \mathbb{N}\}$$
>*Remark*. To allow infinite sums, you need some sort of convergence and you write $\overline{\text{span}}(S)$ to allow convergent infinite sums.

**Concretely**: $\vec v\in\text{ span }(S)\iff$ there are vectors $\vec v_1,\dots ,\vec v_n\in S$ and scalars $a_1,\dots ,a_n\in F$ such that $\vec v=a_1\vec v_1+\dots a_n\vec v_n$. You are in the span if you are a finite sum of vectors in the span.

**Ex.** In $\mathbb{R}^3$, $\text{span}\{\begin{pmatrix}1\\0\\0\end{pmatrix},\begin{pmatrix}0\\1\\0\end{pmatrix}\}=\{a\begin{pmatrix}1\\0\\0\end{pmatrix}+b\begin{pmatrix}0\\1\\0\end{pmatrix}\mid a,b\in\mathbb{R}\}$. This is the $xy-$plane.

**Ex.** In $F[x]$, $\text{span}\{1,x,\dots ,x^n\}=\{a_01+a_1x^1+\dots +a_nx^n\mid a_1\in F\}=P_n(F)$

**Exercise.** Prove that if $W$ is a subspace of $V$ containing some finite set, $\Omega=\{w_1,\dots,w_n\}$, then $\text{span}(\Omega)$ is a subspace of $W$ (meaning that $\text{span}(w_1,\dots,w_n)$ is the smallest subspace containing $\Omega$).
***Proof:*** We want to prove that $\text{span}(\Omega)$ is a subspace in $W$. 
1. Note that $0\in\text{span}(\Omega)$ since taking $0w_1+\dots+0w_n=0\in\text{span}(\Omega)$. 
2. Show that for $w,w'\in\text{span}(\Omega)$, $(w+w')\in\text{span}(\Omega)$.
3. Same for $c(w)\in\text{span}(\Omega)$.

>[!theorem]
>Let $V$ be a vector space and $S\subseteq V$ be a subset, then $\text{span}(S)$ is a subspace of $V$. However, any subspace of $V$, $W\subset V$, that contains $S$ also contains $\text{span}(S)$: $\text{span}(S)\subseteq W$.
>Therefore $\text{span}(S)$ is the smallest subspace containing $S$.

**Proof:** When $S=\emptyset$, $\text{span}(S)=\{\vec 0\}$ is a subspace containing every other subspace. We may assume $S\neq \emptyset$. We check:
1. Zero vector, $\vec 0\in\text{span}(S)$ 
Pick some $\vec v\in S$. Then $\vec 0=0\cdot \vec v$ in a linear combination of vectors in $S$, so $\vec 0\in \text{span}(S)$.

2. Commutative addition, for all $\vec x,\vec y\in \text{span}(S)$, $\vec x+\vec y\in\text{span}(S)$
Suppose $\vec x,\vec y\in\text{span}(S)$. This means we can find $\vec v_1, \dots ,\vec v_n\in S$, $a_1,\dots ,a_n\in F$ with $\vec x=a_1\vec v_1+\dots +a_n\vec v_n$, and  $\vec w_1,\dots ,\vec w_m\in S, b_1,\dots ,b_m\in F$ with $\vec y=b_1\vec w_1+\dots +b_m\vec w_m$. Then $\vec x+\vec y=a_1\vec v_1 + \dots + a_1\vec v_n + b_1 \vec w_1 + \dots + b_m\vec w_m$ is a linear combination in $S$, so $\vec x + \vec y \in \text{span}(S)$.

>[!definition]
>Let $S\subseteq V$. Then $S$ **spans** (or generates) $V$ if $\text{span}(S)=V$.

**Ex.** In $\mathbb{R}^3$, consider $\text{span}(\begin{pmatrix}1\\0\\0\end{pmatrix},\begin{pmatrix}0\\1\\0\end{pmatrix})=\{a_1\begin{pmatrix}1\\0\\0\end{pmatrix}+a_2\begin{pmatrix}0\\1\\0\end{pmatrix}\mid a_1,a_2\in\mathbb{R}\}=\{\begin{pmatrix}a_1\\a_2\\0\end{pmatrix}\mid a_1,a_2\in\mathbb{R}\}$, which is the $xy$-plane.

**Ex.** In $P(F)$, $\text{span}(\{1,x,x^2,\dots ,x^n\})$ in set $\{a_0\cdot 1+\dots +a_nx^n\mid a_i\in F\}$ in the space $P_{\leq n}(F)$ of polynomials with degree $\leq n$.

**Ex.** $S_1$ and $S_2$ are subsets of $V$.
1. Prove $\text{span}(S_1\cap S_2)\subseteq \text{span}S_1\cap \text{span}S_2$
We know that $\text{span}(S_1\cap S_2)\subseteq \text{span}S_1$ and $\text{span}(S_1\cap S_2)\subseteq\text{span}S_2$. Then $\text{span}(S_1\cap S_2)\subseteq\text{span}S_1\cap\text{span}S_2$.

2. $\text{span}(S_1\cup S_2)=\text{span}S_1+\text{span}S_2$
Take any $v\in\text{span}(S_1\cup S_2)$, then $v=a_1v_1+a_2v_2+\dots+a_nv_n+b_1w_1+b_2w_2+\dots+b_mw_m$ for $v_i\in S_1$, $w_i\in S_2$. Because $a_1v_1+\dots+a_nv_n\in\text{span}S_1$ and $b_1w_1+\dots+b_mw_m\in\text{span}S_2$, then $v\in \text{span}S_1+\text{span}S_2$.
For any $w\in\text{span}S_1+\text{span}S_2$, we can write $w=c_1x_1+\dots+c_lx_l+d_1y_1+\dots+d_ky_k$ where $x_i\in S_1$ and $y_i\in S_2$. Then $w\in \text{span}(S_1\cup S_2)$.
