## Pair programming with AI
- Can ask AI to critique code you wrote & for suggestions to improve it
- Make sure you understand and review AI written code
	- Ask AI to explain its code
- Iteratively prompt AI with concrete steps
- Write tests for the AI code, prompt AI to make the test pass, and **refactor AI-written code**

## Which coding tasks are well-suited for AI?
- Boilerplate and framework code
- Repetitive or structural work
- Ideal use case: tasks that are tiresome to solve but easy to verify, and the cost of being wrong is low
### AI is not good for:
- Complex logic paths
- Context-heavy features
- Cognitively demanding work

## Prompt engineering tips
- Keep the task simple
	- AI has limited context size and reasoning capabilities
	- **Break up a task** into smaller & simpler tasks if possible
- Be specific and tell the AI the important **success criteria**
- Can tell the AI to ask you relevant questions before beginning a task
