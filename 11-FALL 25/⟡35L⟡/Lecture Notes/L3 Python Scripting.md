# Python
- **Interpreted** language (no compiler, code is executed line by line at runtime)
	- Slower than compiled code - additional overhead at runtime
- Requires python to be installed on the system
	- C/C++ compiles directly to target platform, doesn't need to be installed
- Variables are **dynamically typed**
	- Hard to follow/debug
- Strings are **immutable** and can't be changed unlike in C++
- Range is **inclusive-exclusive**
