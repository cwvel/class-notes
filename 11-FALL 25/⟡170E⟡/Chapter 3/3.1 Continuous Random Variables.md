Let $X$ denote the outcome of selecting a point randomly from the interval $[a,b]$ where $-\infty<b<\infty$. The probability that $X$ is selected from $[a,x]$, $a<x<b$, is
$$P(a\leq X\leq x)=\dfrac{x-1}{b-a}$$
The cumulative distributive function (cdf)
$$F(x)=P(X\leq x)=\begin{cases}0\hspace{5em}\text{if }x<a \\ \dfrac{x-a}{b-a}\hspace{2em}\text{ if }a\leq x< b\\1\hspace{5em}\text{if }b\leq x\end{cases}$$
can be written as
$$F(x)=\int_{-\infty}^xf(y)dy\hspace{2em}\text{with}\hspace{2em}f(y)=\begin{cases}\dfrac{1}{b-a}\hspace{2em}a\leq y\leq b\\0\hspace{2em}\text{otherwise}\end{cases}$$
So $F'(x)=f(x)$, and $f(x)$ is the probability density function (pdf) of $X$:
$$f(x)=F'(x)=\dfrac{d}{dx}\left(\int_{-\infty}^xf(y)dy\right)$$

>[!definition]
>The **probability density function** (pdf) of a continuous random variable $X$ on a space $S_x$ is an integrable function $f(x)$ such that the following properties hold:
>(i) $f(x)\geq 0,\hspace{2em} x\in S_x$
>(ii) $\int_S f(x)dx=1$
>(iii) if $(a,b)\in S_x$, then $$P(a< X<b)=\int_a^b f(x)dx$$
>****
>To confirm a pdf, check that the function is positive and sums to 1.

***Remarks***
1. If $X$ is CRV with pdf $f(x)$ then
$$P(a\leq X\leq b)=P(a<X\leq b)=P(a<X<b)=P(a\leq X<b)=\int_a^bf(x)dx$$
2. $F(x)=\int_{-\infty}^xf(y)dy,\hspace{1em}F'(x)=f(x)$

## Expected Value and Variance
Consider the function $u(x)$ and CRV $X$.
$$E(u(X))=\int_\mathbb{R}u(x)f(x)dx=\int_{-\infty}^\infty u(x)f(x)dx$$
#### Mean
$$\mu=E(X)=\int_\mathbb{R}xf(x)dx$$

#### Variance
$$\sigma^2=E((X-\mu)^2)=\int_{-\infty}^\infty(x-\mu)^2f(x)dx$$

#### MGF
$$M_X(t)=\int_\mathbb{R}e^{tx}f(x)dx=E(e^{tx})$$

## Uniform Distribution
>[!definition]
>$X\sim\text{Unif}(a,b)$ if the pdf $$f(x)=\begin{cases}\dfrac{1}{b-a}\quad a\leq x\leq b\\0\quad\text{otherwise}\end{cases}$$
>Probability is $\dfrac{1}{b-a}$ everywhere from $a$ to $b$.

#### Mean
$$\mu=E(X)=\int_\mathbb{R}xf(x)dx$$
$$=\int_{-\infty}^a xf(x)dx+\int_a^b xf(x)dx+\int_b^{-\infty}xf(x)dx$$
The outer values cancel out:
$$=\int_a^b xf(x)dx=\int_a^bx\cdot\dfrac{1}{b-a}dx=\dfrac{a+b}{2}$$
So $$\boxed{\mu=E(X)=\dfrac{a+b}{2}}$$
#### Variance
$$\sigma^2(X)=E(X^2)-(E(X))^2)$$
$$E(X^2)=\int_\mathbb{R}x^2f(x)dx=\int_a^b x^2\cdot\dfrac{1}{b-a}dx=\dfrac{1}{3}(b^2+ab+a^2)$$
$$E(X^2)-(E(X))^2=\dfrac{1}{12}(a-b)^2$$

#### MGF
$$M_X(t)=E(e^tx)=\int_\mathbb{R}e^{tx}f(x)dx=\dfrac{1}{b-a}\int_a^be^{tx}dx$$
Simplify
$$M_X(t)=\dfrac{e^{tb}}{(b-a)t}(1-e^{t(a-b)}),\qquad\text{for }t\neq 0$$
If $t=0$ then $M_X(t=0)=1$. Since the MGF is a piecewise function we cannot take the derivatives to find the mean and variance directly.

>[!definition]
>Given $p\in[0,1]$, the 100 $p^\text{th}$ percentile is a number $\Pi_p$ such that
>$$F(\Pi_p)=\int_{-\infty}^{\Pi_p}f(x)dx=p$$
