- flipping a coin: $\mathcal{S}_{H,T}\rightarrow\mathbb{R},x\rightarrow\begin{cases}H\rightarrow\text{real number}\equiv1\\T\rightarrow\text{real number}\equiv0\end{cases}$
- rolling a dice: $\mathcal{S}=\{1,2,\dots,6\}\rightarrow\mathbb{R},x\rightarrow\begin{cases}1\\2\\3\\4\\5\\6\end{cases}$
- Can create a relationship $x:\mathcal{S}\rightarrow\mathbb{R}$. Random variable $x$ is a function from $\mathcal{S}$ to $\mathbb{R}$.

>[!definition]
>Given an outcome space $\mathcal{S}$, a **function** $X$ that assigns one and only one real number $X(s)=x$ to each element $s$ in $\mathcal{S}$ is called a **random variable (RV)**.
>The **space** (range) of $X$ is the set of real numbers $\{x:X(s)=x,s\in S\}$. $S_x$ is also called the **support** of $X$, and contains all the outcomes associated with $X$. When $S$ is discrete, the RV is said to be discrete.

**Ex.** Roll a fair dice: $S=\{1,2,\dots,6\}$
$x:S\rightarrow\mathbb{R}$, $s\rightarrow X(s)=x$
$S_x=\{1,2,3,4,5,6\}$. For each $x\in S_x$, $P(X=x)=\dfrac{1}{6}$.
$\sum_{k\in S_x}P(X=k)=\dfrac{1}{6}+\dots+\dfrac{1}{6}=1$

>[!definition]
>The **probability mass function** (pmf) $f(x)$ of a discrete RV $X$ is a function that satisfies:
>i) $f(x)>0, x\in S_x$
>ii) $\sum_{x\in S}f(x)=1$
>iii) $P(X\in A)=\sum_{x\in A}f(x)$, where $A\subseteq S_x$
>**Remark:** If $x\notin S_x$, then $f(x)=0$ because $P(X=x)=0$

**Ex.** The pmf of $X$ is given by $f(k)=\dfrac{1}{6}$ for $k=1,\dots,6$, if $A=\{1,2,3\}=\text{``X<4''}$, and $A\subseteq S_x$,$$P(X\in A)=\sum_{x\in A}f(x)=\sum_{k=1}\dfrac{1}{6}=\frac{3}{6}=0.5$$

>[!definition]
>A **cumulative distribution function** (cdf) $F(x)$ of a RV $X$ is a function given by
>$$F(x)=P(X\leq x), -\infty<x<\infty$$
>**Remark.** $F$ is called **distribution function.**

**Ex.** PE rolling two dice. The random variable $X$ is the max of 2 rolls.
$S_x=\{(1,2,3,4,5,6)\}$, for $k\in S_x, F(k)=P(X\leq k$)
- $P(X=1)=\dfrac{1}{36}$
- $P(X=2)=\dfrac{3}{36}$
$P(X=k)=\dots$
$\{X=k\}=\{\text{max}(R_1,R_2)=k\}=\{R_1=k,R_2<k\}\cup\{R_1<k,R_2=k\}\cup\{R_1=k,R_2=k\}$
